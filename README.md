# 🎯 Project Overview

This project focuses on analyzing YouTube channel performance using an interactive and visually rich dashboard built in **Google Looker Studio**.

The dashboard helps content creators understand:

✅ Audience Engagement

✅ Video Performance

✅ Watch Time Trends

✅ Subscriber Growth

✅ Content Effectiveness

✅ Sharing Behavior

By converting raw YouTube Analytics data into meaningful visualizations, the dashboard supports data-driven decision making for content strategy.

# 📊 Key Business Insights

## 🚀 Channel Growth

* Views increased significantly during the selected period.
* Positive subscriber growth was observed.
* Traffic spikes indicate successful content publishing windows.

## 🎮 Content Performance

* BGMI and gaming-related content generated the highest engagement.
* Highlight videos consistently outperformed other content categories.

## ⏳ Watch Behavior

* Short-form content delivered higher view counts.
* Audience retention decreases for longer videos.

## 📱 Content Sharing

* WhatsApp emerged as a major sharing source.
* Shared content produced stronger engagement signals.
